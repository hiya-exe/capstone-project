# Week 2 – Service Checklist

This table shows the initial checks we plan to implement for each service.
## RDS

- RDS-001 – Publicly accessible DB
- RDS-002 – Storage encryption disabled
- RDS-003 – Weak backup retention
- RDS-004 – Auto minor version upgrade disabled

## EBS

- EBS-001 – Unencrypted volumes 
- EBS-002 – Encryption by default disabled
- EBS-003 – Snapshots shared too widely
- EBS-004 – Public snapshots of real data
