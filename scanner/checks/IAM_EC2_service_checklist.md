# Week 2 – Service Checklist

This shows the initial checks we plan to implement for each service.

## EC2

- EC2-001 – Public IP + open admin port
- EC2-002 – Unnecessary public IP 
- EC2-003 – Unencrypted EBS volumes
- EC2-004 – SG wide open ports 

## IAM

- IAM-001 – Wildcard `*` policies 
- IAM-002 – Users without MFA 
- IAM-003 – Old/unused access keys 
- IAM-004 – Root account usage
