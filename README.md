# Cloud Security Posture Assessment Tool

## Project Overview

This project focuses on building a cloud security posture assessment tool for AWS. The goal is to identify common cloud security misconfigurations and present the results in a clear dashboard format.

The tool will help detect security issues, map findings to security benchmarks, and provide useful information for improving the cloud environment.

## Project Objective

The main objective of this project is to research AWS services, understand common misconfigurations, and develop a tool that can scan and report security findings.

## Current Project Scope

For the initial phase, the team will research two AWS services and create an AWS account to explore them hands-on. This will help us better understand misconfigurations, service settings, and possible detection methods.

## Key Features

- Research selected AWS services
- Identify common security misconfigurations
- Explore AWS services using a test AWS account
- Develop scanning rules for selected services
- Map findings to security benchmarks
- Display results through a dashboard
- Provide severity levels for findings
- Support future expansion to more AWS services
